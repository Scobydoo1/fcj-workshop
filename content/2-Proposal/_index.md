# 🚀 Project Proposal: SmartHire AI

**Project Name:** SmartHire AI  
**Type:** Cloud-Native AI Recruitment & Intelligent Interview Platform  
**Architecture:** Serverless AWS / React 18 (Vite) / Terraform IaC / PostgreSQL (pgvector)  

## 1. Executive Summary
**SmartHire AI** is a highly scalable, serverless recruitment platform designed to eliminate the bottlenecks of traditional hiring. By leveraging an event-driven AWS architecture, custom Python-based AI processing pipelines, and real-time GraphQL subscriptions, SmartHire automates resume screening, objectively ranks candidates against job descriptions using semantic vector search, and provides an innovative virtual interview workspace equipped with live emotion, speech, and code analysis.

## 2. Problem Statement
Traditional recruitment processes suffer from several critical inefficiencies:
- **Time-Consuming Screening:** HR professionals manually parse unstructured CVs, leading to slow turnaround times and hiring bottlenecks.
- **Subjective Evaluation & Keyword Stuffing:** Standard ATS (Applicant Tracking Systems) rely on exact keyword matches, missing highly qualified candidates who use different terminology. 
- **Static Remote Interviews:** Standard video calls lack behavioral insights, making it difficult to assess candidate confidence, communication skills, or emotional composure.
- **Fragmented Workflows:** Recruiters jump between separate tools for job posting, candidate tracking, live coding tests, and interview recording.

## 3. Proposed Solution & Key Features

### 🎯 1. Frictionless Candidate Portal
- **Secure Authentication:** Passwordless and social login flows powered by Amazon Cognito, managed via global `authStore` states.
- **Direct-to-Cloud CV Uploads:** Utilizes AWS S3 Pre-signed URLs (`CVUploadWidget.tsx`) allowing candidates to upload heavy PDF files directly to S3, completely bypassing API Gateway payload limits and backend bottlenecks.
- **Dynamic Job Board:** Candidates view AI-matched roles in a tailored dashboard, complete with an AI compatibility `MatchScoreGauge`.

### 📊 2. Recruiter Dashboard & Real-Time AI Processing
- **Automated Parsing & Orchestration:** When a CV is uploaded, an S3 event triggers an AWS Step Function (`state_machine_cv_processing.asl.json`). This orchestrates specific Python Lambdas (`cv_parser.py` and `jd_parser.py`) to extract skills, education, and experience.
- **Semantic Vector Matching:** Candidate profiles and Job Descriptions are converted into embeddings and stored in Amazon RDS PostgreSQL using the `pgvector` extension, allowing the system to match candidates based on contextual meaning, not just exact keywords.
- **Real-Time AppSync Updates:** As the AI pipeline (`candidate_ranking_engine`) finishes scoring, AWS AppSync pushes GraphQL subscriptions to the frontend (`useAppSyncSubscription.ts`), updating the recruiter's `CandidateRankings` table instantly without page refreshes.

### 🎙️ 3. Intelligent Interview Workspace
An integrated WebRTC-based conferencing room built for technical and behavioral assessments:
- **Behavioral Insights:** Custom React hooks run locally in the browser to analyze the candidate. `useEmotionAnalysis.ts` tracks facial sentiment, while `useMicVolume.ts` monitors voice confidence and hesitation.
- **Live Code Runner:** A built-in code execution environment (`useCodeRunner.ts`) allows recruiters to conduct live technical assessments without third-party platforms.
- **Automated STAR Evaluation:** Audio is transcribed in real-time (`useSpeechTranscript.ts`) and sent to the `star_evaluator_lambda.py`, which grades the candidate's answers based on the Situation, Task, Action, Result framework. A final report is generated by `report_generator_lambda.py`.

## 4. Technical Architecture & DevOps
SmartHire AI is engineered for enterprise-grade security and extreme scalability:
- **Frontend Layer:** React (TypeScript), Vite, Tailwind CSS, shadcn/ui components, and global Theme context (Dark/Light mode). Hosted on Amazon S3 and distributed globally via Amazon CloudFront.
- **API & Data Layer:** Amazon API Gateway (REST), AWS AppSync (GraphQL), and Amazon RDS (PostgreSQL).
- **Compute & Orchestration Layer:** Fully serverless using AWS Step Functions, Amazon SQS/SNS for decoupled queuing, and AWS Lambda for all AI and backend logic.
- **Infrastructure as Code (IaC):** The entire AWS environment is modularized (Networking, Auth, Database, Processing, Storage) and provisioned using **Terraform**.
- **CI/CD Pipeline:** Fully automated via GitHub Actions (`deploy.yml`). Commits to `main` trigger a build process that securely syncs artifacts to S3 and invalidates CloudFront caches via OIDC authentication.

## 5. Expected Outcomes & Business Value
- **Reduce Time-to-Hire by 70%:** Automated screening allows recruiters to focus their time exclusively on the top-tier candidates.
- **Uncover Hidden Talent:** Semantic search (`pgvector`) ensures great candidates aren't filtered out just because their resume lacks a specific buzzword.
- **Data-Driven Hiring Decisions:** The intelligent interview workspace removes human bias by providing objective metrics on candidate communication, technical skill, and behavioral composure.
